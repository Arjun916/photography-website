# Saikiran VS Nambiar — Portfolio Website

A cinematic portfolio built with React + Vite.

## 🚀 Quick Start

### Prerequisites
- [Node.js](https://nodejs.org/) version 18 or higher

### Run Locally

```bash
# 1. Install dependencies
npm install

# 2. Start development server
npm run dev
```

Then open http://localhost:5173 in your browser.

### Build for Production

```bash
npm run build
```

This creates a `dist/` folder ready to deploy.

## 🌐 Deploy to Vercel (Free)

1. Push this folder to a GitHub repository
2. Go to https://vercel.com and sign in with GitHub
3. Click "Add New Project" → select your repo
4. Click Deploy — you're live!

## 🌐 Deploy to Netlify (Free)

1. Run `npm run build`
2. Go to https://netlify.com
3. Drag & drop the `dist/` folder onto the Netlify dashboard
4. Instant live URL!

## ✏️ Customization

- **src/App.jsx** — All your content, styles, and components live here
- Update your email address in the Contact section (search for `saikiran@example.com`)
- Add real project images by replacing the gradient backgrounds in the `projects` array
- Update social links (Instagram, Vimeo, etc.) in the footer/contact section

